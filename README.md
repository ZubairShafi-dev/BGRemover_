# BGRemover_
🎨 BG Remove – AI Powered Photo Editor

BG Remove is an AI-powered photo editing application that allows users to instantly remove image backgrounds and create professional visuals using a canvas-based editor.
The app is designed for fast, clean, and creative visual creation without requiring advanced design skills.

📱 About the App

BG Remove leverages Google ML Kit’s on-device machine learning to accurately detect and remove backgrounds from images with a single tap.

Once the background is removed, users can place their subject on a custom canvas and enhance it using:

Ready-made templates

Backgrounds & gradients

Text, shapes, and stickers

Social media frames and banners

Every element inside the canvas is fully movable, resizable, rotatable, and customizable, giving users full creative control.

✨ Key Features

🧠 AI Background Removal (Google ML Kit)

🎨 Canvas-Based Photo Editor

🖼️ Import images from gallery

✂️ One-tap background cutout

🧩 Ready-made templates (social posts, posters, banners)

🖌️ Add & edit text, shapes, stickers

🔄 Move, resize & rotate elements

🌈 Backgrounds, gradients & textures

📤 Export high-quality images

📱 Optimized for social media visuals

🛠️ Tech Stack

🎨 Custom Canvas Editor – Drag, scale, rotate elements

🤖 Google ML Kit – On-device background segmentation

⚙️ Kotlin – Native Android development

🗄️ Room Database – Save edits & projects locally

🔥 Firebase – Analytics / optional cloud features
